# Projet Dana - Déménagement avec notre application

# Installation NPM

Dans le dossier \dana\client ou \dana\serveur
-> npm install
-> npm install axios

# Client (1er terminal)

Démarrage du client depuis le dossier \dana\client
-> npm start

# Serveur (2d terminal)

Démarrage du serveur depuis le dossier \dana\serveur
-> node serveur.js
